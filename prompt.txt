Role:
You are a senior full-stack developer and software architect with expertise in PHP, MySQL, web security, bilingual websites, cloud deployment, and safe mental health platforms.

Goal:
Design and implement an MVP of a bilingual (Swahili + English), anonymous, moderated hope-message platform for Tanzania. The platform must prioritize safety, ethical content, scalability, and ease of sharing, and be deployable on Evolution Host (cPanel, PHP + MySQL).

1. Core Requirements

Anonymous message submission

No login required

User selects:

Language: Swahili or English

Category: Life, Faith, Education, Family, Finance, Encouragement, Recovery, etc.

Format: Quote / Paragraph / Lesson Learned

Character limits:

Quote: ≤ 200 chars

Paragraph / Lesson: ≤ 600 chars

Submission triggers pending moderation

Moderation workflow

Admin-only dashboard

Actions:

Approve

Edit (lightly, for safety/clarity)

Reject

Status values: pending / approved / rejected

Audit trail of actions

Viewing & filtering

Filter messages by language and category

Default view shows all approved messages

Messages remain in the original language

Reactions

Support-only reactions (no like counts):

“This helped me”

“Gives me hope”

“You are not alone”

No downvotes, no competitive metrics

Sharing

Share message text only

Platform watermark included

Safe for WhatsApp, Instagram Story

No personal data or sensitive metadata

Bilingual UI

Full site translation (UI text, prompts, buttons)

Language switcher always visible

Translation files: en.php and sw.php

Messages remain in original language

Admin panel also bilingual

Support & resource directory

Hospitals, counselors, NGOs

Consultant contacts for emergencies

Static, verified, clearly labeled

Disclaimer: platform is not a replacement for professional care
